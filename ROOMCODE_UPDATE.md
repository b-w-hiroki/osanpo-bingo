## 技術的な詳細

### ひらがな5文字の生成ロジック

```javascript
generateRoomCode() {
  const hiragana = [
    'あ', 'い', 'う', 'え', 'お',
    'か', 'き', 'く', 'け', 'こ',
    // ... 46文字
  ];
  
  let code = '';
  for (let i = 0; i < 5; i++) {
    const randomIndex = Math.floor(Math.random() * hiragana.length);
    code += hiragana[randomIndex];
  }
  
  return code; // 例: 'さくら'
}
```

### パターン数
- 46文字のひらがなから5文字を選ぶ（重複あり）
- **約2億通り** (46^5 = 205,962,976)
- 重複の心配なし

## 改善案（オプション）

### オプション1: 手動入力も可能にする

現在は読み取り専用ですが、以下の変更で手動入力も可能に:

```html
<input 
  type="text" 
  id="roomCodeInput" 
  class="input-text"
  placeholder="ひらがな5文字"
  maxlength="10"
>
```

**メリット**:
- 友達から教えてもらった合言葉を直接入力できる
- 好きな言葉を使える

**デメリット**:
- 誤入力の可能性

### オプション2: コピーボタンを追加

合言葉を簡単にコピーできるように:

```html
<button id="copyRoomCodeBtn">📋 コピー</button>
```

```javascript
copyRoomCodeBtn.addEventListener('click', () => {
  navigator.clipboard.writeText(roomCode);
  alert('合言葉をコピーしました！');
});
```

### オプション3: QRコード生成

合言葉をQRコードで共有:
- スマホで読み取るだけで参加できる
- 入力不要
