# お題の難易度基準 📊

## 難易度分類の思想

お散歩ビンゴのお題は、**「散歩中に見つかる確率」**と**「場所の限定性」**を基準に3段階に分類しています。

---

## 🟢 かんたん（Easy）- 40個

### 基本思想
**「どこを歩いても、ほぼ確実に見つかるもの」**

### 分類基準

#### 1. 頻出度が極めて高い
- **動物**: ネコ、犬、鳥、人
- **植物**: 花、木、草、葉っぱ
- **自然**: 空、雲、太陽、風、水、影、光
- **建造物**: 家、道、橋、川

#### 2. 場所を選ばない
- 住宅街でも商店街でも公園でも見つかる
- 特定の施設や場所に依存しない

#### 3. 抽象的な概念を含む
- **色**: 赤い物、青い物、黄色い物
- **形**: 丸い物、三角の物、四角の物
- **大きさ**: 大きい物、小さい物
- **感覚**: 音がする物、いい匂い、暖かい場所、涼しい場所

**狙い**: 初心者でも楽しめる、確実に達成感を得られる

### 具体例
```
✅ ネコ → どこでも会える可能性が高い
✅ 花 → 庭、道端、公園、どこにでもある
✅ 空 → 見上げれば必ずある
✅ 赤い物 → ポスト、看板、車など何でもOK
✅ 音がする物 → 車、風、鳥の声など何でもOK
```

---

## 🟡 ふつう（Medium）- 40個

### 基本思想
**「少し注意して探せば見つかるが、場所によっては見つからない可能性もあるもの」**

### 分類基準

#### 1. ある程度の頻出度
- すぐには見つからないが、探せば見つかる
- 散歩コースによって有無が分かれる

#### 2. 特定の用途・場所に関連
- **公共設備**: 郵便ポスト、自販機、コンビニ、バス停
- **交通関連**: 電車、踏切、線路、歩道橋、駐車場、駐輪場
- **街の設備**: 看板、街灯、標識、時計、マンホール、ゴミ箱
- **施設**: 神社、お寺、鳥居、噴水、階段、坂道

#### 3. やや限定的な動植物
- **動物**: カラス、鳩、蝶、虫、アリ
- **植物**: 桜、紅葉、どんぐり、つくし、落ち葉
- **季節限定**: 水たまり、霧、月

**狙い**: 適度な探索の楽しさ、コースによって戦略性が生まれる

### 具体例
```
🔍 郵便ポスト → 住宅街にはあるが、公園では見つからない
🔍 神社 → 近くにあれば見つかるが、ない地域も多い
🔍 桜 → 季節や場所に依存
🔍 カラス → よくいるが、必ずいるわけではない
🔍 マンホール → 道路にあるが、気づかないと見逃す
```

---

## 🔴 むずかしい（Hard）- 40個

### 基本思想
**「レアなもの、特別な場所・条件でしか見つからないもの、運に左右されるもの」**

### 分類基準

#### 1. 極めて低い遭遇率
- **自然現象**: 虹、流れ星
- **特殊車両**: 消防車、救急車、パトカー、キャンピングカー
- **珍しい動物**: リス、ウサギ、カメ、カエル、トカゲ、白鳥、フクロウ

#### 2. 特定の場所・条件が必要
- **大規模な自然**: 海、山、森、湖、滝、洞窟
- **特定施設**: 城、塔、灯台
- **季節限定**: 梅、藤、紫陽花、向日葵、コスモス、彼岸花
- **超レア**: 四つ葉のクローバー

#### 3. 計画的に訪れる必要がある
- 普通の散歩では出会えない
- 特定の場所（動物園、植物園、海岸、山など）に行く必要がある

**狙い**: チャレンジ要素、特別な散歩体験、長期的な目標

### 具体例
```
⭐ 虹 → 雨上がりの特定条件でしか見られない
⭐ 消防車 → 偶然の遭遇か、消防署に行く必要
⭐ リス → 特定の公園や森にしかいない
⭐ 海 → 海岸地域に行く必要がある
⭐ 紫陽花 → 6月の特定の場所のみ
⭐ 四つ葉のクローバー → 運と根気が必要
```

## 📊 難易度別の配分思想

### かんたんモード
```
easy: 24個（100%）
```
- 子供でも楽しめる
- 短時間（30分）でビンゴ達成可能
- どんな場所でも遊べる

### ふつうモード
```
easy: 12個（50%）
medium: 12個（50%）
```
- 適度な探索が必要
- 1時間程度でビンゴ達成を目指す
- コース選択に戦略性

### むずかしいモード
```
easy: 8個（33%）
medium: 8個（33%）
hard: 8個（33%）
```
- 長期的なチャレンジ
- 複数回の散歩で徐々に達成
- 特別な場所への訪問を促進

## 🎯 設計意図

### 1. 達成感のバランス
- **かんたん**: すぐに達成できる → モチベーション維持
- **ふつう**: 適度な難易度 → 飽きない
- **むずかしい**: 長期目標 → やり込み要素

### 2. 散歩の多様性
- 簡単なお題 → 日常の散歩
- 中程度のお題 → コース選択の楽しさ
- 難しいお題 → 新しい場所への冒険

### 3. 年齢層への配慮
- **子供**: かんたんモードで十分楽しめる
- **大人**: ふつうモードで適度な挑戦
- **やり込み派**: むずかしいモードで長期的な目標

### 4. 場所の制約
- **都市部**: easy + medium がメイン
- **郊外**: easy + medium + hard（一部）
- **自然豊か**: 全難易度を楽しめる

## 📝 お題選定の詳細ルール

### ✅ 採用基準

1. **客観性がある**
   - 「ネコ」「花」→ 誰が見ても明確
   - ❌「かっこいい物」→ 主観的すぎる

2. **写真に撮りやすい**
   - 形があるもの、目で見えるもの
   - 「風」は抽象的だが、木の揺れなどで判断可能

3. **安全性**
   - ❌「高い場所」「危険な場所」は除外
   - 子供でも安全に探せるもの

4. **散歩との親和性**
   - 散歩中に自然に出会えるもの
   - ❌「ゲームセンター」など散歩と関係ないものは除外

### ❌ 除外基準

1. **プライバシー侵害**
   - ❌「他人の顔」「家の中」など

2. **危険性**
   - ❌「車道」「崖」など

3. **主観的すぎる**
   - ❌「美しい景色」「感動するもの」など

4. **散歩と無関係**
   - ❌「テレビ」「ゲーム」など屋内のもの

## 🔄 難易度の調整方針

### 今後の改善案

#### 1. ユーザーデータに基づく調整
- 達成率が高いお題 → かんたん
- 達成率が低いお題 → むずかしい
- 実際のプレイデータで微調整

#### 2. 地域別の難易度
- 都市部と郊外で異なる難易度テーブル
- GPSベースで自動選択

#### 3. 季節対応
- 春: 桜、つくし → かんたん
- 冬: 桜 → むずかしい（実質不可能）

#### 4. カスタム難易度
- ユーザーが独自に難易度を設定可能
- 「うちの近所では神社は簡単」など

## 💡 まとめ

### 難易度の核心
```
かんたん: どこでも、いつでも、誰でも
ふつう  : 少し探せば、場所を選べば
むずかしい: 特別な場所、運、計画が必要
```

### バランスの重要性
- 簡単すぎる → 飽きる
- 難しすぎる → 諦める
- **適度な難易度 → 楽しい！**

現在のお題は、この思想に基づいて120個を厳選しています。
