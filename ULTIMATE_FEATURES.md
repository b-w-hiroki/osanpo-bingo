## 🚶 実際の散歩シーンから考える

### 📍 散歩前（準備フェーズ）

#### 1. 今日のおすすめお題 ⭐⭐⭐
**シーン**: 散歩に出る前に「今日は何を探そう？」と考える

**機能**:
- 天気に合わせたお題をハイライト
  - 晴れ: 「太陽」「影」「青空」
  - 雨: 「傘」「水たまり」「紫陽花」
- 季節に合わせたお題を強調
  - 春: 「桜」「つくし」
  - 秋: 「紅葉」「どんぐり」
- 時間帯に合わせて
  - 朝: 「朝日」「通勤電車」
  - 夕方: 「夕焼け」「帰宅する人」

**UI例**:
```
☀️ 今日のおすすめ
・太陽（晴れてます）
・花（春ですね）
・公園（天気が良いので）
```

#### 2. 散歩コース提案 ⭐⭐
**シーン**: 「どこを歩こう？」

**機能**:
- 未達成お題から逆算
- 「公園」が未達成 → 「○○公園に行きませんか？」
- 距離・所要時間の目安
- 過去に行ったコースの記録

**UI例**:
```
🗺️ おすすめコース
駅前コース（20分）
→ 公園、看板、自販機が見つかりそう
```

#### 3. 目標設定 ⭐
**シーン**: 「今日は○個見つけよう」

**機能**:
- 今日の目標を設定（5個マーク、1ビンゴなど）
- 達成したら通知
- 目標に合わせたコース提案

---

### 🚶‍♂️ 散歩中（プレイフェーズ）

#### 4. お題の優先度表示 ⭐⭐⭐
**シーン**: 「何を探せばいいんだっけ？」

**機能**:
- 残りお題だけ表示するフィルタ
- 見つけやすさの色分け
  - 緑: よく見つかる
  - 黄: たまに見つかる
  - 赤: レア
- ビンゴに近いラインを強調

**UI例**:
```
🎯 あと2個でビンゴ！
残り: 看板、公園
[残りお題のみ表示]
```

#### 5. クイック写真撮影 ⭐⭐⭐
**シーン**: 散歩中に素早く撮影したい

**機能**:
- マスをタップ → すぐカメラ起動
- 撮影 → 自動で保存＆マーク
- 複数枚撮影して後で選ぶ
- 連写モード

**UI**:
- ワンタップで即撮影
- プレビューなし（後で確認）

#### 6. 音声フィードバック ⭐⭐
**シーン**: 画面を見ずに散歩したい

**機能**:
- マーク時に音声「ネコ、達成！」
- ビンゴ時に音声「ビンゴ達成！」
- 残りお題を音声で読み上げ

#### 7. お題のヒント ⭐
**シーン**: 「これって○○かな？」と迷う

**機能**:
- お題をタップ → 詳細説明
- 例: 「看板」→「お店の看板、道路標識など」
- 参考写真を表示（他のユーザーの例）

#### 8. 連続達成ボーナス ⭐
**シーン**: 調子が良いときのモチベーション

**機能**:
- 短時間で複数達成 → ボーナスポイント
- 連続コンボ表示「5コンボ！」
- タイムボーナス

## 🎯 「最高のお散歩ツール」の定義

### 目指すべきユーザー体験

**Before（普通の散歩）**:
```
散歩する → 景色を見る → 帰宅 → 終わり
```

**After（お散歩ビンゴを使う）**:
```
アプリを開く
↓
「今日のおすすめ: 花、公園」を見る
↓
散歩に出る
↓
お題を探す → 見つける → 写真を撮る → ポイント獲得
↓
ビンゴ達成！（紙吹雪）
↓
サマリー画面で振り返り
↓
ポイント・バッジ獲得を確認
↓
「明日も散歩しよう！」と思う
↓
習慣化 → 健康維持
```

### 核となる3要素

1. **ゲーミフィケーション**
   - ポイント、レベル、バッジ
   - 達成感・収集欲

2. **記録・振り返り**
   - 履歴、写真、日記
   - 思い出として残る

3. **継続のモチベーション**
   - デイリーチャレンジ、ストリーク
   - 目標設定、通知
