# 合言葉を3-5文字に修正 ✅

## 変更内容

### 旧仕様
- 2-5文字の単語
- 2文字: `あお`、`そら`、`はる` など
- → **短すぎて覚えにくい**

### 新仕様
- **3-5文字の単語のみ**
- 2文字の単語を除外
- より覚えやすく、伝えやすい

## 除外された2文字単語

以下は短すぎるため除外:
- ❌ `あお`（青）
- ❌ `あか`（赤）
- ❌ `もも`（桃）
- ❌ `き`（木）
- ❌ `ひ`（日）
- ❌ `よる`（夜）→ 代わりに `よぞら`（夜空）

## メリット

### 1. 覚えやすさ
- ❌ 旧: `あお` → 短すぎて印象に残らない
- ✅ 新: `みどり` → 適度な長さで覚えやすい

### 2. 伝えやすさ
- ❌ 旧: 「あ、お」→ 一文字ずつ伝える必要
- ✅ 新: 「みどり」→ まとまった言葉として伝えられる

### 3. 聞き取りやすさ
- 電話やビデオ通話でも明確
- 音声認識でも精度が高い

### 4. 誤入力の防止
- 3文字以上なら、入力ミスに気づきやすい

## 実装

### コード
```javascript
generateRoomCode() {
  const words = [
    // 3-5文字の単語リスト
    'きいろ', 'みどり', 'さくら', 'ねこ', 'そら', ...
  ];
  
  // 3文字以上のものだけをフィルタ
  const validWords = words.filter(word => word.length >= 3);
  
  return validWords[Math.floor(Math.random() * validWords.length)];
}
```
