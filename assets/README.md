## 画像生成について

このプロジェクトでは、AI画像生成に対応しています。

### 追加で画像を作りたい場合

以下のような指定が可能です：

- **テーマ**: 散歩、自然、ビンゴ、家族、犬の散歩 など
- **スタイル**: フラット、イラスト、写真風、ミニマル など
- **配色**: プロジェクトカラー（#7eb89a）に合わせる
- **用途**: ヒーロー画像、アイコン、装飾、バナー など

### 例
- 「特徴セクション用の小さなイラストを3つ」
- 「フッター用のソーシャルアイコン」
- 「ビンゴボードのプレビュー画像」

必要な画像があれば、テーマや用途を指定して依頼してください。
