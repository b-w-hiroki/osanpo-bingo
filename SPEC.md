## 1. アイコン

| 項目 | 規格 |
|------|------|
| 解像度 | 512×512px |
| 形式 | PNG（透過） |
| ファイル名 | `icon-{名前}.png`（英小文字・ハイフン） |
| 配置 | `assets/icons/` |
| 図形サイズ | キャンバスの約70～80%に収める |
| トーン | フラット、パステル、かわいい |
| 線 | クリアなアウトライン |

| 表示サイズ（CSS） | PC 4.2rem / スマホ 3.2rem / 小 2.8rem |

## 6. ゲーム機能

| 項目 | 仕様 |
|------|------|
| ボード | 5×5、中央FREE |
| お題数 | 24種 + FREE |

| 写真保存 | LocalStorage（圧縮 JPEG） |

---

## 7. ボタン・操作

| ボタン | 動作 |
|--------|------|
| 作り直す | 確認後、お題をシャッフルして新しいボードを作成（合言葉・難易度は維持） |
| 終わる | 終了確認 → 結果画面（日付・グループ入力 → 決定 → 画像DL・SNS共有） |
| トップへ戻る | index.html へ遷移 |

**結果画面フロー:** 日付・グループ入力 → 決定 → 画像ダウンロード or SNS共有（#お散歩ビンゴ #散歩 #ビンゴ）

## 9. 今後の検討項目（提案）

| 項目 | 内容 |
|------|------|
| 合言葉の結果表示 | 一緒に遊んだ人への伝達用に、結果画面に合言葉を表示 |
| 難易度の結果表示 | どの設定で遊んだか記録に残す |
| 完成度表示 | ○/25マス（例: 15/25） |
| 散歩テーマ | 例: 春の散歩、公園めぐり など自由入力 |
| 複数ハッシュタグ案 | #お散歩ビンゴ #散歩 #ビンゴ #お出かけ など |
