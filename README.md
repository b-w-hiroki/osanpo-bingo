# お散歩ビンゴ

散歩中に見つけたものをマークしてビンゴを目指すゲームです。

## 機能

### 基本機能
- 5×5のビンゴマス（25マス）
- 中央のマスはフリーマス（最初からマーク済み）
- 散歩中に見つけたお題をタップしてマーク
- 縦・横・斜めのいずれかが揃ったらビンゴ達成
- リセット機能（マークのみクリア）
- 新しいゲーム機能（お題をシャッフル）

### 写真機能
- **マスクリック時に写真アップロードを促す**（写真優先のゲームプレイ）
- 写真なしでもマーク可能（スキップオプション）
- 各マスに写真を添付可能（長押し、右クリック、Pキー）
- 写真の自動圧縮（最大800px、容量節約）
- カメラからの直接撮影対応
- 写真のプレビュー・削除機能

### マルチプレイ機能
- 合言葉でグループプレイ（同じ合言葉でも各自異なるシート）
- 決定論的シャッフル（合言葉が同じなら、お題のプールは同じ）
- ユーザーIDによる個別シート生成

### 難易度設定
- **かんたん**: 見つけやすいお題のみ（信号、ネコ、花など）
- **ふつう**: バランスの取れた難易度（おすすめ）
- **むずかしい**: 全種類のお題（海、山、川なども含む）

### アクセシビリティ
- キーボード操作対応（Enter: マーク、P: 写真、Tab: 移動）
- スクリーンリーダー対応（aria-label付き）
- タッチデバイス最適化（スクロール干渉を回避）

### 統計・ポイント機能
- マーク数、写真数、ビンゴ数をリアルタイム表示
- 累計ビンゴ数を記録

### カスタマイズ機能
- 自分でお題を追加・削除可能
- カスタムお題は次回ゲームから反映
- お題リストの管理画面

### データ管理
- **エクスポート機能**（JSON形式でダウンロード）
- **インポート機能**（バックアップから復元）
- 写真を含むすべてのデータを保存・復元

### シェア機能
- ビンゴ画面をスクリーンショットで保存
- SNS共有対応（対応ブラウザのみ）
- 画像として直接ダウンロード

### その他
- PWA対応（オフラインでも利用可能）
- LocalStorageで自動保存
- レスポンシブデザイン

## Cursor上での実行方法

1. **ターミナルでサーバーを起動**  
   プロジェクトフォルダで以下を実行します。
   ```bash
   npx --yes serve -l 3000
   ```
   表示された **http://localhost:3000** をブラウザで開くか、Cursorの **Simple Browser** で開きます。

2. **Simple Browserで開く**  
   - `Ctrl+Shift+P`（Macは`Cmd+Shift+P`）→「Simple Browser: Show」を選択  
   - 表示された入力欄に `http://localhost:3000` と入力（上記サーバー起動後）

3. **Cursorのタスクから起動**  
   - `Ctrl+Shift+P` →「Tasks: Run Task」→「お散歩ビンゴ: サーバー起動」を選択  
   - ターミナルに表示された **http://localhost:3000** のリンクをクリックして開く

4. **ファイルから直接開く**  
   エクスプローラーで `index.html` をダブルクリックして、既定のブラウザで開くこともできます。

## 使い方

1. 上記のどれかでアプリを開く
2. 散歩中にビンゴマスに書かれたお題を見つける
3. 見つけたらマスをタップしてマークをつける
4. 縦・横・斜めのいずれかが揃ったらビンゴ達成！
