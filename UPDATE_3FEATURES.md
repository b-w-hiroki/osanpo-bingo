# 3つの新機能を実装！✨

## ✅ 実装した機能

### 1. 合言葉を5文字以内に制限

**変更内容**:
- 2つの単語の組み合わせ → **単語1つ**に変更
- 5文字以内の単語のみを使用

**生成される合言葉の例**:
- **色**: あお、あか、きいろ、みどり、しろ、くろ、もも
- **自然**: そら、うみ、やま、かわ、もり、いけ、たに
- **植物**: はな、き、くさ、さくら、もも、ばら、すみれ
- **動物**: ねこ、いぬ、とり、さかな、うさぎ、くま、きつね
- **天体**: ほし、つき、ひ、にじ、ひかり
- **季節**: はる、なつ、あき、ふゆ
- **時間**: あさ、ひる、よる
- **天気**: はれ、くもり、あめ、ゆき、かぜ、つゆ
- **場所**: みち、はし、いえ、にわ、もん、まど
- **その他**: ゆめ、うた、おと、いろ、かげ、みず

**単語数**: 約65種類

**メリット**:
- ✅ **覚えやすい** - 5文字以内で短い
- ✅ **伝えやすい** - LINEやメールで簡単
- ✅ **入力しやすい** - 友達が入力する時も楽

---

### 2. 参加人数の表示

**機能**:
- ゲーム設定で**参加人数を入力**（1-99人）
- 統計表示に**👥 人数**を追加
- 「新しいゲーム」で人数を変更可能

**使い方**:
```
1. ゲーム設定モーダルで人数を入力
   例: 3人で遊ぶ → 「3」と入力

2. ゲーム開始後、統計に表示
   👥 人数: 3

3. 人数を変更したい場合
   「新しいゲーム」→ 人数を変更
```

**注意**:
- これは**手動入力**です
- 実際の接続人数ではなく、**参考値**として表示
- リアルタイムの人数表示にはサーバーが必要

---

### 3. 合言葉をタップでコピー

**機能**:
- 統計表示の**🔑 合言葉**をタップ
- クリップボードに自動コピー
- コピー完了の通知を表示

**使い方**:
```
1. 統計の「🔑 合言葉」部分をタップ
   ↓
2. 合言葉がクリップボードにコピーされる
   ↓
3. 画面下に通知
   「📋 合言葉をコピーしました！」
   ↓
4. LINEやメールに貼り付け
```

**視覚的なフィードバック**:
- タップ時に背景色が一瞬変わる（グリーン）
- 2秒間通知が表示される
- ホバー時に少し浮き上がる

## 💻 技術的な実装

### クリップボードAPI
```javascript
copyRoomCode() {
  if (navigator.clipboard && navigator.clipboard.writeText) {
    navigator.clipboard.writeText(this.roomCode)
      .then(() => {
        // コピー成功
        this.showCopyNotification();
      })
      .catch(err => {
        // フォールバック: alert表示
        alert(`合言葉: ${this.roomCode}`);
      });
  }
}
```
