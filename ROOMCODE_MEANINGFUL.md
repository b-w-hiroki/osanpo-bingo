# 合言葉を意味のある言葉に改善 ✨

## 変更内容

### 旧仕様: ランダムなひらがな5文字
```
あいうえお
かきくけこ
さしすせそ
→ 意味がない、覚えにくい
```

### 新仕様: 意味のある言葉の組み合わせ
```
あおいそら
はるのはな
あかいばら
たのしいにじ
→ 意味がある、覚えやすい、伝えやすい
```

---

## 生成ロジック

### 2つの単語を組み合わせ

**前半の言葉**（29種類）:
- **色**: あお、あか、きいろ、みどり、しろ、くろ、ちゃいろ、むらさき、ももいろ
- **形容詞**: おおきな、ちいさな、たのしい、うれしい、きれいな、あかるい、やさしい、かわいい
- **季節**: はる、なつ、あき、ふゆ
- **時間**: あさ、ひる、ゆうがた、よる
- **天気**: はれ、くもり、あめ、ゆき

**後半の言葉**（31種類）:
- **自然**: そら、うみ、やま、かわ、もり
- **植物**: はな、き、くさ、はっぱ、さくら、もも、ばら、たんぽぽ、すみれ、ひまわり、こすもす
- **動物**: ねこ、いぬ、とり、さかな
- **天体**: ほし、つき、たいよう、にじ
- **要素**: かぜ、くも、ひかり、みず
- **場所**: みち、はし、いえ、にわ

**組み合わせ数**: 29 × 31 = **899通り**

## メリット

### 1. 覚えやすい
- ❌ 旧: 「あいうえお」→ 意味がないので忘れやすい
- ✅ 新: 「あおいそら」→ イメージが湧くので覚えやすい

### 2. 伝えやすい
- ❌ 旧: 「あ、い、う、え、お」→ 一文字ずつ伝える必要がある
- ✅ 新: 「あおいそら」→ まとまった言葉として伝えられる

### 3. 散歩にマッチ
- 自然、季節、天気など、散歩に関連する言葉が多い
- お散歩ビンゴの雰囲気にぴったり

### 4. 親しみやすい
- 日常的な言葉で親しみやすい
- 子供でも理解しやすい

友達A: 「今日お散歩ビンゴやろう！」

## 技術的な詳細

### コード
```javascript
generateRoomCode() {
  const firstWords = [
    'あお', 'あか', 'きいろ', ... // 29個
  ];
  
  const secondWords = [
    'そら', 'うみ', 'やま', ... // 31個
  ];
  
  const first = firstWords[Math.floor(Math.random() * firstWords.length)];
  const second = secondWords[Math.floor(Math.random() * secondWords.length)];
  
  return first + second; // 例: 'あおいそら'
}
```

### 文字数
- 最短: 6文字（例: `あおそら`）
- 最長: 13文字（例: `ゆうがたひまわり`）
- 平均: 8-10文字
